# SeaRate - Maritime Shipping Tracking Landing Page

A modern and responsive single-page website for **SeaRate**, a maritime shipping tracking platform. Built with **HTML5**, **CSS3**, and **jQuery**, this landing page showcases ship details, progress indicators, and route tracking in a clean and interactive UI.

## 🚢 Overview
This is a **single-page landing template** designed to showcase the features of **SeaRate**, a platform helping businesses find and track the best maritime transport options. The page includes:

- Live tracking of ships
- Percentage-based progress bars for arrival status
- Detailed ship information cards
- Smooth scroll navigation
- Fully responsive layout

All content is contained within a single HTML file for easy deployment and prototyping.

## 💡 Features
- Interactive ship cards with details
- Real-time progress display (e.g., 78% to destination)
- Clean and modern design
- Fully responsive for all devices
- jQuery-powered animations and interactions

## 💻 Technologies Used
- **HTML5** – Semantic structure and accessibility
- **CSS3** – Responsive layout, transitions, Flexbox/Grid
- **jQuery** – Dynamic UI effects and animations
